# Repair System

A full-stack repair service platform where customers can post repair jobs and vendors can apply, manage assigned jobs, upload profile/shop images, and track payments and reviews.

## Project Overview

This project is a service marketplace for repair work. Customers create repair jobs by adding issue details, budget, address, preferred time, and device images. Vendors browse open jobs in the marketplace, apply with an offer, and update the repair status after selection. The customer can then mark Cash on Delivery as paid and leave a rating and review.
